# RTX 5090 Results

Store RTX 5090 benchmark CSVs here.

Use filenames that identify the model and sweep, for example:

```text
qwopus-coder-mtp-q5-baseline-2026-06-22.csv
qwen-text-nvfp4-vllm-baseline-2026-06-22.csv
```

Required companion context:

- The MSI Afterburner voltage/frequency curve should be applied before long runs.
- The benchmark command should be included in the commit or notes.
- Any crash, thermal shutdown, throttling, or manual fan/curve change should be noted beside the CSV.
